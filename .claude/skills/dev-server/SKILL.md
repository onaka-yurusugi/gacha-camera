# 開発サーバー起動

## 指示

以下のコマンドで開発サーバーを起動:

```bash
npm run dev
```

サーバーは http://localhost:3000 で起動します。

## 確認項目

1. カメラアクセスの許可が必要
2. HTTPS が必要な場合は別途設定が必要
3. モバイルデバイスでテストする場合はネットワーク経由でアクセス

## トラブルシューティング

- ポートが使用中の場合: `lsof -i :3000` で確認
- 依存関係エラー: `npm install` を実行
